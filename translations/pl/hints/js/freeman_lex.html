Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

Proszę złapać jakiś pomysł 💡! To zadanie jest naprawdę proste. Obwód to suma długości wszystkich boków prostokąta. A przeciwległe boki prostokąta mają taką samą długość. To wszystko, czego Państwo potrzebują.

Ok, od czego powinienem zacząć?

Ma Pan length i width, proszę dodać je do siebie. 

length + width
Co robić dalej?

Pozostałe dwa boki mają taką samą długość, więc aby uzyskać obwód, należy pomnożyć poprzednią sumę przez 2.

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcji i zwrócić wynik.

Oto całe rozwiązanie:

function rectanglePerimeter(length: number, width: number): number {

    return (length + width)*2;
}
