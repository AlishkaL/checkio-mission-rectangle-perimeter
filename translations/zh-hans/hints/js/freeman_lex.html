出个主意💡！这个任务其实很简单。周长是矩形所有边长的总和。而矩形的对边具有相同的长度。这就是你需要的一切。

好吧，我应该从哪里开始呢？

您有length 和width ，请将它们相加。 

length + width
下一步该怎么做？

其他两条边的长度相同，因此要得到周长，需要将前面的和乘以 2。

那么，我的最终解决方案应该是怎样的呢？

你只需将代码封装成函数表达式并返回结果即可。

下面是整个解决方案：

function rectanglePerimeter(length: number, width: number): number {

    return (length + width)*2;
}
