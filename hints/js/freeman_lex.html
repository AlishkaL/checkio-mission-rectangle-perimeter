Catch an idea üí°! This mission is actually easy. Perimeter is a sum of lengths of all sides of rectangle. And opposite sides of rectangle have the same length. That's everything you need.

Ok, what should I start from?

You have length and width, add them together. 

length + width
What to do next?

Other two sides have the same length, so to get perimeter you need to multiply previous sum by 2.

So, how my final solution should look like?

You just need to wrap your code into function expression and return the result.

Here is the whole solution:

function rectanglePerimeter(length: number, width: number): number {

    return (length + width)*2;
}
